## General services
This file will share information on how to run all the backend services at once. 
See the individual README.md files in the various folders to run individual services. 

## How to run
### 1. Complete the first two steps of both the Strava README and the peer-server README. 

### 2. Use docker-compose
```bash
docker-compose up
```

### 3. Expose to test with Strava
If you want to test with Strava, follow steps 5 and 6 in the Strava README.
